# 🎬 Netflix Content Analysis Dashboard (Power BI)

## 📌 Project Overview
This project analyzes the Netflix dataset using Microsoft Power BI.

## 📊 Dashboard Features
- Total Titles KPI
- Movies vs TV Shows Distribution
- Content by Country
- Rating Analysis
- Release Year Trend
- Interactive Dashboard

## 🛠 Tools Used
- Power BI
- Power Query
- DAX
- Microsoft Excel/CSV

## 📂 Dataset
Netflix Titles Dataset

## 🚀 Insights
- Total Titles: 8,809
- Movies: 6,131
- TV Shows: 2,676
- United States has the highest number of titles.
- TV-MA is the most common rating.
